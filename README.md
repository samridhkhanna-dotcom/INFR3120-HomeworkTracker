# Homework Tracker  
## INFR3120 – Assignment 3  
Student: Samridh Khanna  

### Overview  
The Homework Tracker is a web application built with Node.js, Express, MongoDB (Atlas), Mongoose, and EJS templating.  
Users can create, view, edit and delete their homework tasks — all secured and stored in MongoDB.

### Features  
- Add homework tasks: title, course, due date, status, description  
- View all tasks in a clean Bootstrap-based list  
- Edit existing tasks  
- Delete tasks with confirmation  
- Responsive design using Bootstrap  
- Secure credentials via .env and .gitignore
